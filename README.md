# 📊 Super Store Dashboard with 15 Day Forecast
 
An interactive Power BI dashboard analyzing Super Store sales data with a 15-day sales forecast using time-series analysis.

---

## 🚀 Project Overview

This dashboard provides insights into:

- 📈 Sales Performance
- 💰 Profit Analysis
- 🛒 Category & Sub-Category Trends
- 🌍 Region-wise Sales Distribution
- 🔮 15-Day Future Sales Forecast

The forecast is generated using Power BI time-series forecasting tools.

---

## 🛠 Tools & Technologies Used

- Power BI
- DAX
- Data Modeling
- Time Series Forecasting
- Excel Dataset
